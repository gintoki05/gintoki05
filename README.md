### AI Microservice for ITSM · `itsm-autocomplete`
> Self-hosted NLP AI engine for Indonesian-language ticket intelligence

- Indonesian NLP model: `LazarusNLP/all-indo-e5-small-v2` (384-dim, self-hosted)
- Features: **Duplicate Ticket Detection**, **Smart Solution Suggestions**, **Semantic Knowledge Search**, **Technician Action Suggestions**, **Service Mismatch Detection**, **Automated Problem Detection**
- Stack: **FAISS** vector index + **PostgreSQL pgvector** for hybrid search

`FastAPI` `Sentence Transformers` `pgvector` `FAISS` `PyTorch` `APScheduler` `SQLAlchemy`
